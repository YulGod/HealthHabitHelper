Contributing Guide
==================

Instructions on how to contribute to the HealthHabitHelper project.

1. Bug Reports and Feature Requests
-----------------------------------

Please use the GitHub Issues tab to report bugs or propose new features.

2. Code Contributions
---------------------

1.  Create a feature branch from the ``main`` branch.
2.  Commit your changes. (Use imperative, short commit messages).
3.  Create a Pull Request (PR) to the ``main`` branch and request a review.

3. Community Documentation
--------------------------

Please refer to the [CODE_OF_CONDUCT.md] and [CONTRIBUTING.md] files.
