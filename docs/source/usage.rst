Usage Guide
===========

This section covers the core functionality of HealthHabitHelper.

1. Ultra-Simple Daily Log
-------------------------

Record your mood, energy, and sleep quality quickly using minimal input on the main screen. The goal is to spend no more than 30 seconds on logging.

2. Concise Food Log
-------------------

Log meals using simple text entry, without the burden of calorie counting or detailed nutritional analysis.

3. Logging Habit Tracker
------------------------

Visualize your logging streaks and receive positive feedback to encourage daily consistency.
